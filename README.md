# Hello World App

A programming console and course application. (WIP)

## Installation

### Using Bun (Recommended)
```bash
bun install
```

## Build

### Using Build Script
```bash
bun build-dist
```

## Run

After building, serve the application:

```bash
bun serve-dist
```

Then open your browser to the provided URL (typically `http://localhost:8000`).
